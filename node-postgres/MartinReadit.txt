Installation zur Einbindung von NodeJS, AngularJS(HTML), PostgreSQL mit Express


1. Installiere NodeJS durch herunterladen v0.12.7

2. npm install -g express-generator@4

3. Gehe in File "node-postgres"

4. npm install

5. Starte Server mit "npm start"
